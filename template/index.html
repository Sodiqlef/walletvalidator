How does Chain Synchronizer work?

There are three main approaches,

  • First we assume the wallet is on a complete network node
  • Secondly, Wallets are securely connected to DAPPS protocol trusted server
  • Lastly, various wallets sync are in direct interaction with the rest of the network node which is the most efficient using the simplified payment verification (SPV) method.

Which wallets do Chain Synchronizer support?

A crypto wallet is a device or application that stores a collection of keys and can be used to send, receive and track ownership of cryptocurrencies.

Chain Sychronizer supports several types of wallets, for the majority of users we recommend using decentralized wallets which hosts a growing array of decentralized apps (dAPPS).
